# Lohnsystem – Brutto-/Netto-Auswertung mit Schichtplanung

Dieses Projekt ist ein webbasiertes Lohnsystem zur Verwaltung von Schichten, Bruttolohn-/Nettolohn-Auswertung und Abwesenheiten. Es beinhaltet eine moderne Benutzeroberfläche, mehrsprachige Darstellung (Deutsch/Englisch) sowie ein konfigurierbares Admin-Panel.

## ✨ Funktionen

- 🔐 Login & Registrierung mit Passwort-Reset
- 👤 Benutzerprofil inkl. Steuerklasse, Freibeträge, Kirchensteuer
- 🕒 Manuelle Überstundenverwaltung
- 💼 Adminbereich für Lohngruppen, Zuschläge und Feiertagskonfiguration
- 📅 Schichteinträge (Früh/Spät/Nacht), Abwesenheiten, Feiertagsberechnung
- 🌍 Mehrsprachig (Deutsch & Englisch)
- 📄 Exportfunktionen (PDF, Excel)
- 📱 Responsives Design – optimiert für Desktop und Mobilgeräte
- 🛡️ Datenschutzkonform (Admins haben keinen Zugriff auf persönliche Daten)

## 🚀 Projekt starten (für Web-Demo)

Du kannst die `index.html` direkt im Browser öffnen oder das Projekt mit einem lokalen Server hosten.

## 🛠️ Projekt starten (Next.js lokal)

```bash
git clone https://github.com/dein-benutzername/lohnsystem.git
cd lohnsystem
npm install
npm run dev
```

## 🗂️ Verzeichnisstruktur

- `index.html` – Weboberfläche (Login/Registrierung)
- `pages/`, `api/` – Backend und Frontend bei Verwendung mit Next.js
- `locales/` – Sprachdateien
- `Admin_Policy_Lohnsystem.md` – Zugriffsmatrix / Datenschutzkonzept
